# Tecnologías requeridas 

Python / R / Bash scripting
SQL + cloud data warehouses (Snowflake, Databricks, Redshift, BigQuery)
Power BI, Tableau o Looker
Herramientas de modelado de datos y capas de reporting

Construcción y mantenimiento de dashboards y reportes
Definición y seguimiento de KPIs y métricas
Pruebas A/B y análisis causal
Validación de calidad de datos
Modelado de datos y buenas prácticas de analytics
Documentación de métricas y procesos

# Habilidades blandas (Soft Skills)

Comunicación escrita y verbal clara (para audiencias técnicas y no técnicas)
Atención al detalle y pensamiento analítico
Trabajo colaborativo en equipos cross-funcionales
Curiosidad e interés por el aprendizaje continuo
Orientación al cliente (mentalidad de consultoría)

#  Responsabilidades principales

Traducir preguntas de negocio en requerimientos analíticos
Identificar tendencias, patrones e insights en los datos
Construir y mantener dashboards de métricas clave
Colaborar con ingenieros de datos y científicos de datos
Presentar hallazgos mediante visualizaciones y reportes
Identificar mejoras en flujos de trabajo y calidad de datos
